# Setup :gear:

## 1. Requirements

* `Python >= 3.8`
* [`pipenv`](https://pipenv-es.readthedocs.io/es/latest/)

## 2. Install dependencies and bootstrap the application :hammer:

```shell
make start
make server-up
```

## 2. Discover another useful commands :mag_right:

```shell
make help
```
